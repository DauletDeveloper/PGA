Password Generator App (PGA)A mobile application built with React Native and Expo designed for secure password generation and local data persistence. The application provides users with a reliable tool to create complex passwords based on customizable criteria and store them directly on their Android device.Core FunctionalityThe application features a robust password generation engine that allows users to adjust length and toggle specific character sets including uppercase letters, numbers, and special symbols. For data persistence, the project implements React Native Async Storage, ensuring that saved passwords remain available across application sessions without the need for an external database or internet connection.Technical ImplementationThe development stack includes React Native for cross-platform UI components and Expo for an optimized development workflow on Android. State management is handled through React hooks, while Async Storage manages the asynchronous serialization and retrieval of password objects. The interface is styled using a modern utility-first approach to ensure a clean and responsive user experience.Installation and DeploymentTo run the project locally, clone the repository and navigate to the frontend directory. Execute npm install to download the necessary dependencies. Start the development server using the npx expo start command. The application can be viewed on a physical Android device via the Expo Go app or through an Android Virtual Device.Project StructureThe codebase is organized into functional modules for maintainability. The src/pages directory contains the primary application screens, while src/components houses reusable UI elements. Logic related to cryptographic generation and storage operations is located within the utility folder to separate business logic from the presentation layer.
